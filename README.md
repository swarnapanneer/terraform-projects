# 🚀 AWS Flask App Deployment using Terraform

## 📌 Project Overview

This project demonstrates how to deploy a Python Flask application on AWS EC2 using Terraform with automated provisioning.

## 🛠️ Tech Stack

* AWS EC2
* Terraform
* Python (Flask)
* Linux

## ⚙️ Features

* Infrastructure as Code using Terraform
* Automated EC2 provisioning
* Flask app deployment
* Production-ready server 

## 📂 Project Structure

```
aws-flask-terraform/
├── app/
├── terraform/

```

## 🚀 Deployment Steps

### 1. Clone Repository

```
git clone https://github.com/your-username/project-folder
cd project-folder
```

### 2. Initialize Terraform

```
terraform init
```

### 3. Apply Configuration

```
terraform apply
```

### 4. Access Application

```
//<EC2-Public-IP>
```
